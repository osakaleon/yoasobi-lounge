About This Server

夜に集まる、風活好きたちのコミュニティ

X やその他の媒体では、凍結やアカウントバンが心配で
思ったことが自由に言えない——そんな経験、ありませんか？

「夜遊び雑談ラウンジ」は、風活好きが集まるクローズドな Discord コミュニティ。
外では言いにくい話も、ここではもう少し自由に、気兼ねなく話せる場所を目指しています。

仲間と気軽に雑談したり、リアルな情報を交換したり。
夜の時間をもっと充実させたい方、ぜひ一緒に盛り上がりましょう。

How To Join

Discordとは？　参加方法

Discord（ディスコード）は、テキスト・画像のやり取りができる無料のコミュニティアプリです。
招待リンクから参加する仕組みで、X とは独立した環境でやり取りが可能。
スマートフォン・PC どちらからでも無料で利用できます。

✅ アカウントあり
すでに Discord をお使いの方
1下の「Discord に参加する」ボタンをタップ
2Discord アプリが開いたら、そのまま参加をタップ
3サーバー内で画面上部のサーバー名をタップ →「サーバープロフィールを編集」を選択
4「ニックネーム」を X と同じニックネームに変更して保存
5続けて「サーバーアバターを変更」から X と同じアイコン画像を設定して完了。
※すでにアイコンを設定済みの方は変更不要です。そのままご参加ください。
🆕 アカウントなし
Discord アカウントをお持ちでない方
1App Store / Google Play で「Discord」を検索してインストール
2アプリを開き「登録」からアカウントを新規作成
3登録完了後、下の「Discord に参加する」ボタンをタップ
4プロフィール設定でニックネームを X と同じ名前に設定
5アイコン画像も X と同じ画像にして完了
Join Conditions

入会条件

安心して楽しめるコミュニティを守るため、以下の条件を設けています

01

X（旧Twitter）で風活に関するアカウントを持っていること
風活に関する内容を定期的に投稿している、アクティブな X アカウントをお持ちの方。非公開アカウントは対象外です。

02

X での投稿がアクティブであること
直近で風活に関する投稿を行っているアカウントが対象です。長期間投稿がない方はご遠慮ください。

03

Discord 内のニックネームを X のニックネームと統一できること
コミュニティの一体感を保つため、Discord のニックネームを X で使用しているニックネームと同じにしていただきます。

04

入会には審査があります
入会申請後、X のアカウントおよび投稿内容を確認させていただきます。コミュニティの趣旨に沿っているかを判断し、参加の可否をご連絡いたします。

🚫 以下の方は入会をお断りします
✕ X が鍵垢（非公開アカウント）の方
✕ X が風活に関係のないアカウントの方
※ プロフィールまたは投稿に風活に関する書き込みがあればOKです
